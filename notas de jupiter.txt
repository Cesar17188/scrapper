Generalmente si se tiene el anacando navigator se debe iniciar con el cmd.Prompt
windows
para crear un ambiente virtual en anaconda: conda create --name "nombre" "librerias"
para activar el ambiente virtual es: conda activate "nombre"
para salir del ambiente es: conda deactivate
para listar los ambientes virtuales: conda env list
para borrar un ambiente: conda remove --name "nombre"
si queremos borrar todos los paquetes del ambiente virtual: conda remove --name "nombre" --all
linux o mac
para crear un ambiente virtual en anaconda: conda create --name "nombre" "librerias"
para activar el ambiente virtual es: source activate "nombre"
para salir del ambiente es: source deactivate
para listar los ambientes virtuales: conda environments
para borrar un ambiente: conda remove --name "nombre"
si queremos borrar todos los paquetes del ambiente virtual: conda remove --name "nombre" --all
